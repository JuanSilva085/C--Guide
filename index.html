Fundamentos

Variáveis e Tipos
// Tipos básicos
int numero = 10;
double decimal = 10.5;
string texto = "Olá, mundo!";
bool verdadeiro = true;
char caractere = 'A';

// Outros tipos comuns
long numeroGrande = 1234567890L;
float pontoFlutuante = 10.5f;
decimal valorPreciso = 10.5m;
C# é uma linguagem fortemente tipada. Cada variável deve ser declarada com um tipo específico antes de ser usada.
Estruturas Condicionais
// If-else simples
if (numero > 0) {
    Console.WriteLine("Número positivo");
} 
else if (numero < 0)
 {
    Console.WriteLine("Número negativo");
} else {

}

// Operador ternário
string resultado = (idade >= 18) ? "Adulto" : "Menor";
Estruturas condicionais permitem que seu código tome decisões baseadas em condições lógicas.
Laços de Repetição
// Loop for
for (int i = 0; i < 10; i++) {
    Console.WriteLine(i);
}


// Loop while
int j = 0;
while (j < 10) {
    Console.WriteLine(j);
    j++;
}

// Loop foreach
foreach (var item in lista) {
    Console.WriteLine(item);
}
Loops permitem executar um bloco de código repetidamente, economizando tempo e reduzindo a duplicação de código.
Arrays e Coleções
// Arrays
int[] numeros = new int[5];
numeros[0] = 10;

// Array inicializado
string[] nomes = { "Ana", "João", "Maria" };

// Lista
List clientes = new List();
clientes.Add("Cliente 1");

// Dictionary
Dictionary idades = new Dictionary();
idades.Add("João", 25);
C# oferece diferentes maneiras de armazenar e manipular coleções de dados, desde arrays simples até estruturas mais complexas.
Switch
// Exemplo de switch
switch (diaDaSemana) {
    case 1:
        Console.WriteLine("Domingo");
        break;
    case 2:
        Console.WriteLine("Segunda-feira");
        break;
    default:
        Console.WriteLine("Dia inválido");
}
O switch avalia uma variável e executa o bloco correspondente ao caso que combina com o valor.
Do While
// Exemplo de do-while
int contador = 0;
do {
    Console.WriteLine(contador);
    contador++;
} while (contador < 5);
O laço do-while executa o bloco de código pelo menos uma vez antes de verificar a condição.
Listas e Foreach
// Exemplo de lista e foreach
List<string> frutas = new List<string> { "Maçã", "Banana", "Laranja" };
foreach (string fruta in frutas) {
    Console.WriteLine(fruta);
}
Listas são coleções dinâmicas em C#. O foreach permite percorrer cada item em uma coleção.
Tratamento de Exceções com Try-Catch
// Exemplo de try-catch para manipulação de erros
try {
    int resultado = 10 / 0; // Gera exceção
} 
catch (Exception ex) {
    Console.WriteLine($"Erro: {ex.Message}");
}
O bloco try-catch captura exceções para evitar que erros interrompam a execução do programa.

Programação Orientada a Objetos

Classes e Objetos
// Definição de classe
public class Pessoa {
    // Propriedades
    public string Nome { get; set; }
    public int Idade { get; set; }
    
    // Construtor
    public Pessoa(string nome, int idade) {
        Nome = nome;
        Idade = idade;
    }
    
    // Método
    public void Apresentar() {
        Console.WriteLine($"Olá! Sou {Nome} e tenho {Idade} anos.");
    }
}
Classes são os blocos fundamentais da programação orientada a objetos em C#, servindo como modelo para criar objetos.
Herança
// Classe base
public class Animal {
    public string Nome { get; set; }
    
    public void Comer() {
        Console.WriteLine("Comendo...");
    }
}

// Classe derivada
public class Cachorro : Animal {
    public void Latir() {
        Console.WriteLine("Au au!");
    }
}
Herança permite que uma classe adquira os atributos e métodos de outra classe, promovendo reuso de código.
Interfaces
// Definição de interface
public interface IVeiculo {
    void Acelerar();
    void Frear();
    int VelocidadeAtual { get; }
}

// Implementação de interface
public class Carro : IVeiculo {
    private int _velocidade = 0;
    
    public int VelocidadeAtual => _velocidade;
    
    public void Acelerar() {
        _velocidade += 10;
    }
    
    public void Frear() {
        _velocidade = Math.Max(0, _velocidade - 10);
    }
}
Interfaces definem contratos que as classes devem implementar, permitindo polimorfismo sem herança direta.
Métodos de Extensão
// Classe de extensão
public static class StringExtensions {
    // Método de extensão para string
    public static bool IsNullOrEmpty(this string str) {
        return string.IsNullOrEmpty(str);
    }
    
    public static string Capitalize(this string str) {
        if (str.IsNullOrEmpty()) return str;
        return char.ToUpper(str[0]) + str.Substring(1);
    }
}

// Uso
string texto = "hello";
string capitalizado = texto.Capitalize(); // "Hello"
Métodos de extensão permitem adicionar funcionalidades a tipos existentes sem modificar a classe original ou criar uma nova classe derivada.
Classes e Objetos com Getters e Setters
// Definição de classe com propriedades e métodos
public class Pessoa {
    private string _nome;
    
    public string Nome {
        get => _nome;
        set => _nome = value;
    }

    public void Apresentar() {
        Console.WriteLine($"Olá, meu nome é {Nome}.");
    }
}

// Criando um objeto:
Pessoa pessoa = new Pessoa();
pessoa.Nome = "João";
pessoa.Apresentar(); // Saída: Olá, meu nome é João.
Classes são modelos para criar objetos. Getters e setters controlam o acesso às propriedades.
Métodos com Retorno e Void
// Método com retorno
int Somar(int a, int b) {
    return a + b;
}

// Método void (sem retorno)
void ExibirMensagem(string mensagem) {

Métodos encapsulam blocos de código reutilizáveis. Eles podem retornar valores ou não (void).

LINQ

Operações Básicas
// Filtrar (Where)
var maioresDeIdade = pessoas.Where(p => p.Idade >= 18);

// Ordenar (OrderBy)
var ordenadosPorNome = pessoas.OrderBy(p => p.Nome);

// Selecionar (Select)
var nomes = pessoas.Select(p => p.Nome);

// Pular e pegar (Skip, Take)
var paginado = pessoas.Skip(10).Take(5);
LINQ (Language Integrated Query) permite escrever consultas diretamente no código C# para trabalhar com várias fontes de dados.
Operações de Agregação
// Contar (Count)
int total = numeros.Count();
int pares = numeros.Count(n => n % 2 == 0);

// Soma (Sum)
int soma = numeros.Sum();
double mediaIdades = pessoas.Average(p => p.Idade);

// Mínimo e Máximo
int menorNumero = numeros.Min();
int maiorIdade = pessoas.Max(p => p.Idade);

// Sintaxe de consulta
var resultado = from p in pessoas
               where p.Idade > 30
               orderby p.Nome
               select new { p.Nome, p.Idade };
LINQ oferece diversas operações de agregação para análise de dados, como contagem, soma, média, mínimo e máximo.
Coleções Genéricas: List e Dictionary
// Trabalhando com List:
List<int> numeros = new List<int>();
numeros.Add(10);
numeros.Add(20);

Console.WriteLine($"Primeiro número: {numeros[0]}"); // Saída: Primeiro número: 10.

// Trabalhando com Dictionary:
Dictionary<int, string> paises = new Dictionary<int, string>();
paises.Add(1, "Brasil");
paises.Add(2, "Argentina");

Console.WriteLine(paises[1]); // Saída: Brasil.
Coleções genéricas como List e Dictionary permitem armazenar dados organizados e acessá-los eficientemente.
